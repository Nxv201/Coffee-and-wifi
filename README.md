# Coffee-and-wifi
Mục tiêu: Tạo một trang web đánh giá các quán cafe
<br>
Các chức năng:
  + Hiện danh sách các quán cafe và đánh giá
  + Có một lối ẩn "/add" cho phép quản trị viên thêm các quán cafe
  + Dữ liệu được lưu trong file CSV
<br>
Các kĩ năng:
  + Flask
  + WTForms
  + Bootstrap
  + Xử lý file CSV
